## BASIC SECTION

### 1. count_occurrences
**Description:**  
You are given a list of values. Count how many times each value appears and return the result as a dictionary.

### 2. remove_duplicates
**Description:**  
You are given a list that may contain duplicate values. Remove duplicates while preserving the original order of elements.

### 3. format_price_list
**Description:**  
You are given a list of prices. Produce a formatted string where each price is displayed with exactly two decimal places.


## INTERMEDIATE SECTION

### 4. split_by_type
**Description:**  
You are given a mixed list containing integers and strings. Separate them into different lists inside a dictionary.

### 5. recursive_max
**Description:**  
You are given a list of numbers. Determine the maximum value using recursion instead of built-in functions.

### 6. transpose_matrix
**Description:**  
You are given a 2D matrix. Return a new matrix where rows become columns.
